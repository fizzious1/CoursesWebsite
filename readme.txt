In order to run the code locally follow the steps:
1. Clone the project from Git to your favorite IDE
2. Edit the 'Run/Debug Configurations as follow:
package.json: ~\WhereYouSavedTheProject\reactproj\package.json
Command: run
Scripts: start
3. Install dependencies from the console. the command is 'npm install'
4. Start the development server. The command is 'npm start'
